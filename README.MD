#  Flight Price Prediction ML Project
## 📌 Overview
This project is a Machine Learning web application that predicts flight ticket prices based on input features like airline, source, destination, date, time, and duration.
##  Features
- Predict flight ticket prices
- User-friendly web interface
- Real-time predictions
- Machine Learning model integration
##  Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- HTML/CSS
##  Machine Learning Model
- Algorithm used: Random Forest / XGBoost / Linear Regression /SVM
- Data preprocessing: Encoding, feature engineering
- Model saved using joblib
